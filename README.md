# Amazon Clone (Tutorial Project)

A web app clone of Amazon built using HTML, CSS, JavaScript, and localStorage.

## Features
- Add to cart and remove items
- Select delivery options for each product
- Real-time payment summary calculation
- Quantity updates per product

## My Enhancements
- Added default delivery option to prevent shipping errors
- Fixed quantity selector to calculate totals correctly on first click
- Improved UI highlighting for selected delivery options

## What I Learned
- DOM manipulation for dynamic UI
- Event handling in JavaScript
- Managing state with localStorage
- Debugging and fixing real-world bugs
